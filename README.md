Exploratory Data Analysis (EDA) Project
Netflix Movies & TV Shows Dataset and Iris Dataset

--------------------------------------------------

PROJECT OVERVIEW

This project focuses on performing Exploratory Data Analysis (EDA) on two datasets:

1. Netflix Movies and TV Shows Dataset
2. Iris Dataset

The purpose of this project is to understand data patterns, feature behavior, and relationships between variables using visualization and statistical techniques.

--------------------------------------------------

OBJECTIVES

- To analyze and visualize important features in the datasets
- To understand data distribution and trends
- To detect outliers and anomalies
- To identify features useful for prediction
- To summarize meaningful insights from the data

--------------------------------------------------

TOOLS & TECHNOLOGIES USED

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

--------------------------------------------------

EDA TECHNIQUES APPLIED

- Histogram plots for numerical feature distribution
- Count plots for categorical features
- Box plots for outlier detection
- Correlation heatmaps for feature relationships
- Feature importance identification

--------------------------------------------------

KEY INSIGHTS

IRIS DATASET:
- Petal length and petal width are the most important features for classification
- Strong correlation between petal length and petal width
- Dataset is balanced across all species
- No major outliers found

NETFLIX DATASET:
- Movies dominate over TV Shows on Netflix
- USA produces the highest number of titles
- Most content was released after 2015
- TV-MA and TV-14 are the most common ratings
- Movie duration varies widely
